(four directions
on each the player changes to state of being pushed backwards when push X
part of the inconvenience is that you can't do anything else when beaming just stop it with X, so you have to set up first and then fire the beam
PlayerState is moving backwards until it hits something
if hits wooden box, pushes that too, but only one
if this metal box, stops
if hits wall, stops
if hits pit, falls down


in the meantime a beam is propagated forwards in a straight line
then it hits something
if hits wooden box, pushes that too, but only one
if hits metal box, stops
if hits wall, stops
(if hits mirror change direction)

======
RULES
======

(---cardinal controls---)
up [ > PlayerBeamlessD ] -> [ > PlayerBeamlessU ]
up [ > PlayerBeamlessL ] -> [ > PlayerBeamlessU ]
up [ > PlayerBeamlessR ] -> [ > PlayerBeamlessU ]

down [ > PlayerBeamlessU ] -> [ > PlayerBeamlessD ]
down [ > PlayerBeamlessL ] -> [ > PlayerBeamlessD ]
down [ > PlayerBeamlessR ] -> [ > PlayerBeamlessD ]

left [ > PlayerBeamlessU ] -> [ > PlayerBeamlessL ]
left [ > PlayerBeamlessD ] -> [ > PlayerBeamlessL ]
left [ > PlayerBeamlessR ] -> [ > PlayerBeamlessL ]

right [ > PlayerBeamlessU ] -> [ > PlayerBeamlessR ]
right [ > PlayerBeamlessD ] -> [ > PlayerBeamlessR ]
right [ > PlayerBeamlessL ] -> [ > PlayerBeamlessR ]

(---recoil pushes crates---)
[ > PlayerBeamless | Crate ] -> [ > PlayerBeamless | > Crate ] 
up [ BeamStartD | PlayerBeamPushedD | Crate ] -> [ > BeamStartD | > PlayerBeamPushedD | > Crate ]
[ > CrateRecoil | obstacleforplayer ] -> [ CrateRecoil | obstacleforplayer ]
down [ > PlayerBeamPushedU | Crate ] -> [ > PlayerBeamPushedU | > Crate ]
left [ > PlayerBeamPushedL | Crate ] -> [ > PlayerBeamPushedL | > Crate ]
right [ > PlayerBeamPushedR | Crate ] -> [ > PlayerBeamPushedR | > Crate ]

(---beams push crates---)
up [ BeamU | Crate ] -> [ BeamU | > Crate ]
down [ BeamD | Crate ] -> [ BeamD | > Crate ]
left [ BeamL | Crate ] -> [ BeamL | > Crate ]
right [ BeamR | Crate ] -> [ BeamR | > Crate ]

(---generally firing the beam---)
up [ action PlayerBeamlessU | No obstacleforlight ] -> [ PlayerBeamPushedU | BeamStartU ] sfx0
down [ action PlayerBeamlessD | No obstacleforlight ] -> [ PlayerBeamPushedD | BeamStartD ] sfx0
left [ action PlayerBeamlessL | No obstacleforlight ] -> [ PlayerBeamPushedL | BeamStartL ] sfx0
right [ action PlayerBeamlessR | No obstacleforlight ] -> [ PlayerBeamPushedR | BeamStartR ] sfx0

(---firing when close to a something---)
up [ no obstacleforplayer | action PlayerBeamlessU | Wall ] -> [ > PlayerBeamPushedU | > BeamStartU | Wall ] sfx0
down [ no obstacleforplayer |  action PlayerBeamlessD | Wall ] -> [ > PlayerBeamPushedD | > BeamStartD | Wall ] sfx0
left [ no obstacleforplayer |  action PlayerBeamlessL | Wall ] -> [ > PlayerBeamPushedL | > BeamStartL | Wall ] sfx0
right [ no obstacleforplayer |  action PlayerBeamlessR | Wall ] -> [ > PlayerBeamPushedR | > BeamStartR | Wall ] sfx0

(---no beams when stopped firing---)
[ PlayerBeamless ] [ Beams ] -> [ PlayerBeamless ] [ ]

([ action PlayerBeamPushedU ] -> [ PlayerBeamlessU ]
[ action PlayerBeamPushedD ] -> [ PlayerBeamlessD ]
[ action PlayerBeamPushedL ] -> [ PlayerBeamlessL ]
[ action PlayerBeamPushedR ] -> [ PlayerBeamlessR ])


(---light is blocked by obstacles---)
late up [ BeamStartU | No ObstacleForLight ] -> [ BeamStartU | BeamU ]
late up [ BeamU | No ObstacleForLight ] -> [ BeamU | BeamU ] again

late down [ BeamStartD | No ObstacleForLight ] -> [ BeamStartD | BeamD ]
late down [ BeamD | No ObstacleForLight ] -> [ BeamD | BeamD ] again

late left [ BeamStartL | No ObstacleForLight ] -> [ BeamStartL | BeamL ]
late left [ BeamL | No ObstacleForLight ] -> [ BeamL | BeamL ] again

late right [ BeamStartR | No ObstacleForLight ] -> [ BeamStartR | BeamR ]
late right [ BeamR | No ObstacleForLight ] -> [ BeamR | BeamR ] again

(---recoiling player---)
up [ BeamStartD | PlayerBeamPushedD | No ObstacleForPlayer ] -> [ > BeamStartD | > PlayerBeamPushedD | ] again
down [ BeamStartU | PlayerBeamPushedU | No ObstacleForPlayer ] -> [ > BeamStartU | > PlayerBeamPushedU | ] again
left [ BeamStartR | PlayerBeamPushedR | No ObstacleForPlayer ] -> [ > BeamStartR | > PlayerBeamPushedR | ] again
right [ BeamStartL | PlayerBeamPushedL | No ObstacleForPlayer ] -> [ > BeamStartL | > PlayerBeamPushedL | ] again

(late left [ BeamStartL | No ObstacleForLight ] -> [ BeamStartL | BeamH ]
late left [ BeamH | No ObstacleForLight ] -> [ BeamH | BeamH ] )

([ Beams ] -> [ ])


(---recoiled player can be changed back to normal one---)
(amíg pushed, addig nem player
de amikor stationary, i.e. megállt
akkor átvált player characterre)
[ stationary PlayerBeamPushedU ] -> [ PlayerStoppedRecoilU ]
[ stationary PlayerBeamPushedD ] -> [ PlayerStoppedRecoilD ]
[ stationary PlayerBeamPushedL ] -> [ PlayerStoppedRecoilL ]
[ stationary PlayerBeamPushedR ] -> [ PlayerStoppedRecoilR ]

[ > PlayerStoppedRecoilU ] -> [ stationary PlayerStoppedRecoilU ]
[ > PlayerStoppedRecoilD ] -> [ stationary PlayerStoppedRecoilD ]
[ > PlayerStoppedRecoilL ] -> [ stationary PlayerStoppedRecoilL ]
[ > PlayerStoppedRecoilR ] -> [ stationary PlayerStoppedRecoilR ]

[ action PlayerStoppedRecoilU ] -> [ PlayerBeamlessU ]
[ action PlayerStoppedRecoilD ] -> [ PlayerBeamlessD ]
[ action PlayerStoppedRecoilL ] -> [ PlayerBeamlessL ]
[ action PlayerStoppedRecoilR ] -> [ PlayerBeamlessR ]

(---falling into pits---)
late [ PlayerBeamless Pit ] -> [ Pit ] sfx1
(animate fall, end of animation sfx2)
(late [ PlayerBeamless Pit ] -> [ PlayerFallsStart Pit ] sfx1
late [ PlayerFalls1 Pit ] -> [ PlayerFalls2 Pit ] 
late [ PlayerFalls2 Pit ] -> [ PlayerFalls3 Pit ] 
late [ PlayerFalls3 Pit ] -> [ PlayerFalls4 Pit ] 
late [ PlayerFalls4 Pit ] -> [ Pit ] sfx2)
